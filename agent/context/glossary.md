# D&D 2024 Glossary

Domain-specific terminology for D&D 2024 rules.

## Core Mechanics

| Term | Definition |
|------|------------|
| **d20** | Twenty-sided die, core resolution mechanic |
| **Ability Check** | d20 + ability modifier + proficiency (if proficient) vs DC |
| **Saving Throw** | d20 + ability modifier + proficiency (if proficient) vs DC |
| **Attack Roll** | d20 + ability modifier + proficiency vs AC |
| **DC (Difficulty Class)** | Target number for checks/saves (5=easy, 10=medium, 15=hard, 20=very hard) |
| **AC (Armor Class)** | Target number for attack rolls to hit |

## Advantage/Disadvantage

| Term | Definition |
|------|------------|
| **Advantage** | Roll 2d20, take the higher result |
| **Disadvantage** | Roll 2d20, take the lower result |
| **Straight Roll** | Roll 1d20 (advantage and disadvantage cancel) |

## Abilities

| Ability | Abbreviation | Primary Use |
|---------|--------------|-------------|
| **Strength** | STR | Melee attacks, athletics, carrying capacity |
| **Dexterity** | DEX | Ranged attacks, AC, initiative, stealth |
| **Constitution** | CON | Hit points, concentration saves |
| **Intelligence** | INT | Arcana, history, investigation |
| **Wisdom** | WIS | Perception, insight, medicine |
| **Charisma** | CHA | Persuasion, deception, intimidation |

## Combat

| Term | Definition |
|------|------------|
| **Initiative** | d20 + DEX modifier, determines turn order |
| **Action** | Main activity on your turn (Attack, Cast Spell, Dash, etc.) |
| **Bonus Action** | Quick activity, only if ability grants one |
| **Reaction** | Response to trigger, one per round |
| **Movement** | Distance you can move on your turn (usually 30 ft) |
| **Opportunity Attack** | Reaction attack when enemy leaves your reach |
| **Flanking** | Optional rule: advantage when ally opposite enemy |

## Damage & Conditions

| Term | Definition |
|------|------------|
| **Hit Points (HP)** | Health, 0 HP = unconscious |
| **Temporary HP** | Buffer HP, doesn't stack |
| **Resistance** | Take half damage from a type |
| **Vulnerability** | Take double damage from a type |
| **Immunity** | Take no damage from a type |
| **Condition** | Status effect (prone, restrained, stunned, etc.) |

## Spellcasting

| Term | Definition |
|------|------------|
| **Spell Slot** | Resource to cast spells, regained on rest |
| **Cantrip** | Spell that doesn't use a slot |
| **Concentration** | Maintaining a spell, broken by damage/other spells |
| **Ritual** | Cast without slot, takes 10 extra minutes |
| **Upcasting** | Using higher slot for increased effect |
| **Spell Save DC** | 8 + proficiency + spellcasting ability modifier |
| **Spell Attack** | d20 + proficiency + spellcasting ability modifier |

## Resting

| Term | Definition |
|------|------------|
| **Short Rest** | 1 hour, spend Hit Dice to heal |
| **Long Rest** | 8 hours, regain all HP, half Hit Dice, spell slots |
| **Hit Dice** | d6/d8/d10/d12 depending on class, for healing |

## Class Features (Common)

| Term | Class | Definition |
|------|-------|------------|
| **Sneak Attack** | Rogue | Extra damage with finesse/ranged weapon + advantage or ally adjacent |
| **Rage** | Barbarian | Bonus damage, resistance, advantage on STR |
| **Smite** | Paladin | Expend slot for extra radiant damage |
| **Wild Shape** | Druid | Transform into beast |
| **Spellcasting** | Multiple | Access to spell lists |
| **Extra Attack** | Fighter/etc | Make additional attacks with Attack action |

## Abbreviations

| Abbrev | Meaning |
|--------|---------|
| RAW | Rules as Written (literal text) |
| RAI | Rules as Intended (designer intent) |
| DM | Dungeon Master |
| PC | Player Character |
| NPC | Non-Player Character |
| PHB | Player's Handbook |
| DMG | Dungeon Master's Guide |
| MM | Monster Manual |
